Some More Limits

$$ \lim_{x \to 0} \frac{ \sqrt{x + 1} - 1 }{ \sqrt[3]{x} - 1 } \\ = \lim_{x \to 0} \frac{ ( \sqrt{x + 1} - 1 )( \sqrt{x + 1} + 1 ) }{ ( \sqrt[3]{x} - 1 ) ( \sqrt{x + 1} + 1 )} \\ = \lim_{x \to 0} \frac{ ( \sqrt{x + 1} )^2 - 1^2 }{ ( \sqrt[3]{x} - 1 ) ( \sqrt{x + 1} + 1 )} \\ = \lim_{x \to 0} \frac{ x + 1- 1 }{ ( \sqrt[3]{x} - 1 ) ( \sqrt{x + 1} + 1 )} \\ = \lim_{x \to 0} \frac{ x }{ ( \sqrt[3]{x} - 1 ) ( \sqrt{x + 1} + 1 )} \\ = \lim_{x \to 0} \frac{ 0 }{ ( \sqrt[3]{0} - 1 ) ( \sqrt{0 + 1} + 1 )} \\ = \lim_{x \to 0} \frac{ 0 }{ -2 } \\ = 0^- $$

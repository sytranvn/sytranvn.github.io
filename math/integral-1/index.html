Integral

$$ I= \int_{-\infin}^{-1}{\frac{dx}{x^2}} \ = \lim_{t \to -\infin}\int_{t}^{-1}{\frac{dx}{x^2}} (1) \ A = \int_{t}^{-1}{\frac{dx}{x^2}} \ = -\frac{1}{x}\Big|{t}^{-1} = 1 + \frac{1}{t} \ \ I = \lim{t \to -\infin}{1 + \frac{1}{t}} \ = 1 + 0 = 0 \square $$

$$ I = \int_{0}^{+\infin}{(x+1).e^{-x}dx} \ = \lim_{0 \to t}\int_{0}^{t}{(x+1).e^{-x}dx} \ A = \int_{0}^{t}{(x+1).e^{-x}dx} \ \text{Let: } \ u = x+1\ dv = e^{-x}dx \ \rArr du = 1dx \ \rArr \text{pick } v = -e^{-x} \ A = uv|0^t - \int{0}^{t}{vdu}|_0^t \ = (x + 1)(-e^{-x})|0^t - \int{0}^{t}{-e^{-x}dx} \ = (t + 1)(-e^{-t}) - (0 + 1)(-e^{-0}) - (-e^{-x}|0^t) \ = (t + 1)(-e^{-t}) - (0 + 1)(-e^{-0}) - (-e^{-t} -e^0) \ = (t + 1)(-e^{-t}) + 1 + e^{-t} - 1 \ = (t + 1)(-e^{-t}) + e^{-t} \ = -te^{-t} \ I = \lim{0 \to t}{-te^{-t}} $$

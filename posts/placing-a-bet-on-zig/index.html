Placing a Bet on Zig

April 16, 2024 One-minute read
Recently, Zig caught a lot of attentions. I'm interested in programming languages, therefore I gave Zig a try.

After zig 0.12.0 is out. I ran into trying new changes and here’s some things I found interesting.

Zig fetch Link to heading

zig fetch is a new way to manage dependencies with zig. In previous version, we need to manually update the build.zig.zon file and then run zig build to be able to install dependencies. Now things get a little bit easier since we can do it in one command zig fetch --save. We also don’t need to muanually edit the checksum hash as well.
However, fetch is only working with tar.gz archives, we can not use arbitrary github repo urls or .zip archives. There’s a tracking issue to add zip archives support.
